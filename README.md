# 📌 Featured Project  
## Boston Housing Regression Optimization

### 🎯 Objective
To improve baseline linear regression performance on the Boston Housing dataset using feature engineering and regularization techniques.

---

## 🧠 Problem Statement

Initial Linear Regression model achieved:

- R² ≈ 0.73

The objective was to systematically improve model performance through structured enhancements.

---

## 🔬 Optimization Strategy

The following improvements were implemented:

- Log transformation of the target variable (MEDV)
- Feature scaling using StandardScaler
- Polynomial feature expansion (degree = 2)
- Ridge Regression (L2 Regularization)
- Hyperparameter tuning for optimal alpha value
- Evaluation on original price scale

---

## 📊 Final Model Performance

| Metric | Value |
|--------|--------|
| R² Score | **0.83** |
| MSE | 12.35 |
| RMSE | 3.51 |

The optimized model explains **83% of the variance** in housing prices.

## 📈 Key Takeaways

- Feature engineering significantly improves model performance.
- Polynomial features capture non-linear relationships effectively.
- Ridge regularization controls overfitting when feature dimensions increase.
- Proper evaluation requires converting predictions back to original scale when using log transformation.
- Optimization is not only about algorithms — it is about structured pipeline design.
